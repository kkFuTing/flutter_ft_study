# Excel模板生成说明

## 方法1：使用Dart脚本生成（推荐）

### 步骤：

1. **运行脚本**
   ```bash
   dart run scripts/generate_excel_template.dart
   ```

2. **生成的文件**
   - 文件名称：`示例题目.xlsx`
   - 文件位置：项目根目录

3. **使用文件**
   - 用Excel打开 `示例题目.xlsx`
   - 可以修改、添加题目
   - 在应用中导入使用

## 方法2：手动转换CSV为Excel

### 步骤：

1. **打开CSV文件**
   - 用Excel打开项目根目录下的 `示例题目.csv`

2. **另存为Excel格式**
   - 点击"文件" → "另存为"
   - 选择保存位置
   - 文件类型选择：**Excel 工作簿 (*.xlsx)**
   - 点击"保存"

3. **完成**
   - 现在你有了Excel格式的文件
   - 可以在应用中导入使用

## 方法3：手动创建Excel模板

### 步骤：

1. **打开Excel**
   - 创建新的工作簿

2. **设置表头**（第一行）
   ```
   题干 | 题型 | 选项A | 选项B | 选项C | 选项D | 答案 | 解析 | 分类 | 标签
   ```

3. **填写示例数据**（从第二行开始）
   - 参考 `示例题目.csv` 中的内容
   - 或者参考 `导入格式说明.md` 中的格式要求

4. **保存文件**
   - 保存为 `.xlsx` 格式

## Excel文件字段说明

### 必需字段：
- **题干**：题目的主要内容
- **答案**：正确答案

### 可选字段：
- **题型**：单选题/多选题/判断题/填空题
- **选项A/B/C/D**：选项内容
- **解析**：题目解析说明
- **分类**：题目分类/章节
- **标签**：用分号或逗号分隔的标签

### 图片字段（可选）：
- **题干图片**：本地路径或网络URL
- **选项A图片/B图片/C图片/D图片**：选项对应的图片
- **解析图片**：解析对应的图片

## 注意事项

1. **编码**：确保Excel文件使用UTF-8编码
2. **表头**：第一行必须是表头，字段名要准确
3. **答案格式**：
   - 单选题：`A`
   - 多选题：`A,B,C`（用逗号分隔）
   - 判断题：`A`（正确）或 `B`（错误）
   - 填空题：直接填写答案内容
4. **图片路径**：
   - 本地路径：`C:/images/question1.jpg`
   - 网络URL：`https://example.com/image.jpg`

## 快速开始

最简单的方式：
1. 运行 `dart run scripts/generate_excel_template.dart`
2. 打开生成的 `示例题目.xlsx`
3. 修改或添加题目
4. 在应用中导入
