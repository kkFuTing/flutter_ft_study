# Flutter 学习指南

## 📚 学习路径（按顺序学习）

### 第一阶段：基础概念（第1-2周）

#### 1. 理解Flutter核心概念
- **Widget是什么**：Flutter中一切都是Widget
  - StatelessWidget：无状态组件
  - StatefulWidget：有状态组件
- **MaterialApp 和 Scaffold**：应用的骨架
- **热重载（Hot Reload）**：开发时的神器

#### 2. 基础Widget学习
从 `lib/examples/01_basic_widgets.dart` 开始学习：
- Text：文本显示
- Container：容器
- Row/Column：横向/纵向布局
- Image：图片显示
- Icon：图标
- Button：按钮（ElevatedButton, TextButton, OutlinedButton）

### 第二阶段：布局和样式（第3-4周）

#### 3. 布局Widget
学习 `lib/examples/02_layouts.dart`：
- **Row & Column**：线性布局
- **Stack**：层叠布局
- **Expanded & Flexible**：弹性布局
- **Padding & Margin**：内边距和外边距
- **Center & Align**：对齐方式
- **SizedBox**：尺寸控制
- **ListView**：列表滚动
- **GridView**：网格布局

#### 4. 样式和主题
- Colors：颜色使用
- TextStyle：文本样式
- ThemeData：主题配置
- 使用装饰（Decoration）美化组件

### 第三阶段：交互（第5-6周）

#### 5. 用户交互
学习 `lib/examples/03_interactions.dart`：
- **StatefulWidget 和 setState**：状态管理基础
- **GestureDetector**：手势检测
- **表单输入**：TextField, TextFormField
- **点击事件**：onPressed, onTap
- **页面导航**：Navigator.push/pop

### 第四阶段：进阶（第7-8周及以后）

#### 6. 路由和导航
- 命名路由
- 路由参数传递
- 底部导航栏（BottomNavigationBar）
- 抽屉导航（Drawer）

#### 7. 状态管理
- Provider
- GetX
- Riverpod
- Bloc

#### 8. 网络请求
- http 或 dio 包
- JSON 序列化
- API 调用

#### 9. 数据持久化
- SharedPreferences（简单数据）
- sqflite（本地数据库）
- Hive（NoSQL数据库）

## 🎯 实践建议

### 每日练习
1. **修改现有代码**：尝试修改 `main.dart` 中的样式、颜色、文字
2. **运行示例代码**：学习 `lib/examples/` 目录下的示例
3. **尝试热重载**：
   - 修改代码后按 `Ctrl+S`（或 `Cmd+S`）
   - 观察变化立即出现在模拟器/设备上

### 第一个小项目建议
完成基础学习后，可以尝试做一个：
1. **待办事项（Todo）应用**
   - 添加任务
   - 删除任务
   - 标记完成
   - 列表显示

2. **天气查询应用**
   - 输入城市名
   - 调用天气API
   - 显示天气信息

## 💡 学习技巧

1. **边学边做**：看完一个知识点，立即写代码实践
2. **修改示例**：不要只复制粘贴，尝试修改参数看效果
3. **理解原理**：理解为什么这样写，而不是死记硬背
4. **遇到问题**：善用搜索引擎和官方文档
5. **做笔记**：记录常用Widget的用法和参数

## 🚀 快速开始

1. 运行项目：
   ```bash
   flutter run
   ```

2. 查看示例：
   - 修改 `main.dart` 中的 `home` 属性
   - 例如：`home: BasicWidgetsExample()`
   - 保存后热重载即可看到效果

3. 学习顺序：
   ```
   main.dart（默认计数器） 
   → 01_basic_widgets.dart（基础组件）
   → 02_layouts.dart（布局）
   → 03_interactions.dart（交互）
   ```

## ⚠️ 常见问题

### Q: 修改代码后没有看到变化？
A: 尝试热重载（Hot Reload），如果还不行，尝试热重启（Hot Restart）

### Q: 不知道某个Widget怎么用？
A: 查看官方文档，或按 `Ctrl+点击` Widget名称查看源码注释

### Q: 报错怎么办？
A: 
1. 仔细看错误信息
2. 检查括号是否匹配
3. 检查导入的包是否正确
4. 运行 `flutter pub get` 安装依赖
