# Excel/CSV 导入格式说明

## 文件格式
- 支持 **Excel** 格式：`.xlsx`、`.xls`
- 支持 **CSV** 格式：`.csv`

## 字段说明

### 必需字段（必须填写）

| 字段名 | 说明 | 示例 |
|--------|------|------|
| **题干** | 题目的主要内容 | Flutter是由哪个公司开发的？ |
| **答案** | 正确答案 | A（单选题）或 A,B,C（多选题） |

### 可选字段（可以不填）

| 字段名 | 说明 | 示例 | 备注 |
|--------|------|------|------|
| **题型** | 题目类型 | 单选题/多选题/判断题/填空题 | 默认为单选题 |
| **选项A** | 第一个选项 | Google | 单选题、多选题、判断题需要 |
| **选项B** | 第二个选项 | Facebook | 单选题、多选题、判断题需要 |
| **选项C** | 第三个选项 | Microsoft | 可选 |
| **选项D** | 第四个选项 | Apple | 可选 |
| **解析** | 题目解析说明 | Flutter是由Google开发的... | 可选 |
| **分类** | 题目分类/章节 | Flutter基础 | 可选 |
| **标签** | 标签（用分号或逗号分隔） | Flutter;基础 | 可选 |

### 图片字段（可选）

| 字段名 | 说明 | 示例 |
|--------|------|------|
| **题干图片** | 题干对应的图片 | C:/images/q1.jpg 或 https://example.com/img.jpg |
| **选项A图片** | 选项A的图片 | C:/images/option_a.jpg |
| **选项B图片** | 选项B的图片 | C:/images/option_b.jpg |
| **选项C图片** | 选项C的图片 | C:/images/option_c.jpg |
| **选项D图片** | 选项D的图片 | C:/images/option_d.jpg |
| **解析图片** | 解析对应的图片 | C:/images/analysis.jpg |

**图片支持格式：**
- 本地文件路径：`C:/images/question1.jpg`
- 网络URL：`https://example.com/image.jpg`

## 题型说明

### 1. 单选题
- **题型**：填写"单选题"或"single"
- **答案**：填写单个选项，如 `A`、`B`、`C`、`D`
- **示例**：
  ```
  题干: Flutter是由哪个公司开发的？
  选项A: Google
  选项B: Facebook
  选项C: Microsoft
  选项D: Apple
  答案: A
  ```

### 2. 多选题
- **题型**：填写"多选题"或"multiple"
- **答案**：填写多个选项，用逗号分隔，如 `A,B,C`
- **示例**：
  ```
  题干: 以下哪些是Flutter的状态管理方案？
  选项A: Provider
  选项B: GetX
  选项C: Bloc
  选项D: Redux
  答案: A,B,C
  ```

### 3. 判断题
- **题型**：填写"判断题"或"judgment"
- **选项A**：通常填写"正确"
- **选项B**：通常填写"错误"
- **答案**：填写 `A`（正确）或 `B`（错误）
- **示例**：
  ```
  题干: Flutter使用Dart语言开发。
  选项A: 正确
  选项B: 错误
  答案: A
  ```

### 4. 填空题
- **题型**：填写"填空题"或"fill"
- **不需要填写选项**（选项A/B/C/D留空）
- **答案**：直接填写正确答案内容
- **示例**：
  ```
  题干: 在Flutter中，使用______关键字可以创建无状态组件。
  答案: StatelessWidget
  ```

## 字段名称别名

系统支持多种字段名称，以下都可以使用：

| 标准字段 | 别名 |
|---------|------|
| 题干 | 题目、stem |
| 答案 | answer |
| 题型 | 类型、type |
| 选项A | A、option_a |
| 选项B | B、option_b |
| 选项C | C、option_c |
| 选项D | D、option_d |
| 解析 | analysis |
| 分类 | 章节、category |
| 标签 | tags |
| 题干图片 | 题目图片、stem_image |
| 选项A图片 | A图片、option_a_image |
| 解析图片 | analysis_image |

## 使用步骤

1. **准备Excel/CSV文件**
   - 第一行填写表头（字段名）
   - 从第二行开始填写题目数据

2. **打开应用**
   - 进入刷题工具
   - 点击"导入题目"

3. **选择文件**
   - 点击"选择文件"按钮
   - 选择你的Excel或CSV文件

4. **预览数据**
   - 点击"预览"按钮
   - 查看解析结果和错误信息

5. **导入题目**
   - 确认无误后点击"导入题目"
   - 等待导入完成

## 注意事项

1. **编码问题**：CSV文件建议使用UTF-8编码
2. **图片路径**：确保图片路径正确，或使用网络URL
3. **答案格式**：
   - 单选题：单个字母，如 `A`
   - 多选题：多个字母用逗号分隔，如 `A,B,C`
   - 判断题：`A`（正确）或 `B`（错误）
   - 填空题：直接填写答案内容
4. **必填字段**：题干和答案必须填写，其他字段可选
