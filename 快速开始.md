# 🚀 快速开始

## 第一步：运行项目

在终端中运行以下命令：

```bash
flutter run
```

或者如果你使用的是VS Code或Android Studio，直接点击运行按钮即可。

---

## 第二步：查看示例

运行项目后，你会看到一个**示例列表页面**，包含三个学习示例：

1. **示例1：基础Widget** - 学习最常用的基础组件
2. **示例2：布局Widget** - 学习如何布局和排列组件
3. **示例3：交互示例** - 学习用户交互和状态管理

点击任意一个卡片，就可以查看对应的示例代码和效果！

---

## 第三步：修改代码学习

### 方法1：修改示例代码
1. 打开 `lib/examples/` 目录下的任意示例文件
2. 修改参数或添加新组件
3. 保存文件（`Ctrl+S`）
4. 在模拟器中点击**热重载**按钮（或按 `r` 键）
5. 立即看到变化！

### 方法2：修改主页面
如果想改回原来的计数器示例：

打开 `lib/main.dart`，找到第35行，改成：

```dart
home: const MyHomePage(title: 'Flutter Demo Home Page'),
```

如果想查看示例列表，改回：

```dart
home: const ExampleListPage(),
```

## 💡 热重载 vs 热重启

### 热重载（Hot Reload）
- 快速：保存后立即更新
- 保持状态：不会重置应用数据
- 使用方法：按 `r` 键或点击热重载按钮

### 热重启（Hot Restart）
- 完全重启：应用从头开始
- 重置状态：所有数据回到初始值
- 使用方法：按 `R` 键或点击热重启按钮

**建议**：日常开发用热重载，遇到奇怪问题时用热重启。

## 📝 学习小技巧

1. **多看注释**：示例代码中都有详细的中文注释
2. **多动手**：不要只看，一定要自己修改代码
3. **多实验**：尝试改变参数，看看会发生什么
4. **查文档**：遇到不理解的Widget，查看[官方文档](https://docs.flutter.dev/)

---

## ❓ 遇到问题？

1. **代码报错**：检查括号是否匹配，导入是否正确
2. **看不到变化**：尝试热重启而不是热重载
3. **不知道某个参数的作用**：在IDE中按 `Ctrl+点击` Widget名称，查看源码和注释
4. **需要帮助**：查看 `学习指南.md` 获取更多资源
